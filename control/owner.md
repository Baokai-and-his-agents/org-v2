## ✅ Owner 自动批准清单

以下动作 Agent 可自动执行，无需请示：

**文档和索引**：
- 创建/更新任务文档
- 更新能力资产索引
- 创建工作记录（logs, heartbeats）
- 修正错别字和链接

**低风险开发**：
- 本地测试
- Lint 和格式化
- 创建草稿 PR
- 更新依赖（非生产）

**研究和报告**：
- 外部情报收集
- 竞品分析
- 技术调研
- 周报生成

---

## 🚫 必须请示 Owner

**资金相关**：
- 任何花钱、签约、购买
- 修改定价、退款政策

**生产操作**：
- 生产发布
- 数据库变更
- 删除数据

**对外承诺**：
- 代表品牌发言
- 法律文件
- 客户沟通（非标准流程）

**权限变更**：
- 修改 secrets
- 添加/删除协作者
- 修改仓库权限

**周五**（复盘）：

## 🔧 资源供给

Agent 需要资源时，在 `ops/resource-requests.md` 创建请求。

## 📈 成功指标

**月度检查**：
- 能力资产数量趋势
- 任务完成 vs 能力沉淀比率
- Agent 自主完成率
- 每周工作时长（Owner vs Agent）

目标：Owner 时间逐月下降，能力资产逐月上升
