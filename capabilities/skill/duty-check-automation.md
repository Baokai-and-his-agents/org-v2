# Skill: Duty Check Automation

**类型**：skill  
**创建**：2026-06-13  
**更新**：2026-06-13  
**复用次数**：10+

## 🎯 用途

自动化日常巡检，检查系统健康状态、能力质量、Git 状态等。

## 📥 输入

- 系统当前状态

## 📤 输出

- 检查报告（通过/警告/错误）
- 具体问题列表
- 建议行动

## 🔄 步骤

```bash
bash ops/scripts/duty-check.sh
```

## 📋 检查项

1. Git 状态（未提交文件）
2. 能力质量（格式检查）
3. 文档完整性
4. 工具可执行性
5. 系统健康度

## 💡 注意事项

- 每日运行（cron 或手动）
- 发现问题立即修复
- 可集成到 CI
