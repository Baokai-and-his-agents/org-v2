# Skill: GitHub Repository Setup

**类型**：skill  
**创建**：2026-06-13  
**更新**：2026-06-13  
**复用次数**：1

## 🎯 用途

从零设置 GitHub 仓库，包含 Issues/Discussions/Actions 配置。

## 🔄 步骤

1. **创建仓库** (public/private)
2. **配置 Settings**:
   - Issues 模板
   - PR 模板
   - Branch 保护规则
3. **启用功能**:
   - Discussions
   - Actions
   - Projects (可选)
4. **添加 README/LICENSE**
5. **设置 CI/CD**

## 💡 注意事项

- Branch protection: 要求 review
- Actions: 质量门禁
- Issues 模板: 降低沟通成本
