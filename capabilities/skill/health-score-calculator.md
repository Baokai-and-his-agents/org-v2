# Skill: Health Score Calculator

**类型**：skill  
**创建**：2026-06-13  
**更新**：2026-06-13  
**复用次数**：5+

## 🎯 用途

计算系统健康度（8 个维度），自动生成改进建议。

## 📥 输入

- 系统当前状态（文件、脚本、配置）

## 📤 输出

- 8 维度评分（0-10）
- 总体健康度评分
- 改进建议

## 🔄 步骤

```bash
bash ops/scripts/calculate-health-score.sh
```

## 📊 评分维度

1. 结构清晰度
2. 可执行性
3. 文档完整度
4. 自动化程度
5. 能力沉淀率
6. 度量可见性
7. 质量保障
8. 外部验证

## 💡 注意事项

- 自动评分 + 人工调整
- 建议保持 ≥ 8.0
- 可自定义维度
