# System: Quality Check Pipeline

**类型**：system  
**创建**：2026-06-13  
**更新**：2026-06-13  
**复用次数**：100+

## 🎯 用途

CI/CD 质量检查流水线设计，确保代码质量。

## 🔄 流水线阶段

1. **能力质量检查** - 检查能力文件格式
2. **测试运行** - 运行单元测试
3. **健康度评估** - 计算系统健康度
4. **门禁判断** - 失败则阻塞合并

## 💡 关键设计

- 阻塞式门禁（失败 = 不能合并）
- 快速反馈（< 2 分钟）
- 详细报告（具体哪里失败）
