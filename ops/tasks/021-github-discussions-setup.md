# 第二十一个任务：设置 GitHub Discussions

**创建**：2026-06-13  
**状态**：进行中  
**类型**：社区建设型  
**优先级**：高  
**预计时间**：20 分钟

---

## 🎯 目标

启用并配置 GitHub Discussions，建立社区交流渠道，为外部验证做准备。

---

## 📋 背景

准备公开发布前，需要：
- 提供用户讨论空间
- 分类管理不同话题
- 降低 Issues 噪音
- 建立社区氛围

---

## 📊 任务范围

### In Scope
- 启用 Discussions
- 创建分类（Q&A、Ideas、Show and tell、General）
- 置顶欢迎帖
- 添加到 README/CONTRIBUTING

### Out of Scope
- 大量内容填充
- 复杂的 bot 集成
- 自定义主题

---

## 🔄 执行计划

### 1. 启用功能
- [ ] GitHub Settings → Features → Discussions
- [ ] 确认分类设置

### 2. 创建分类
- [ ] Q&A（问答）
- [ ] Ideas（功能建议）
- [ ] Show and tell（展示）
- [ ] General（常规讨论）

### 3. 置顶帖
- [ ] 欢迎新成员
- [ ] 社区准则
- [ ] 常见问题快速链接

### 4. 文档更新
- [ ] README 添加链接
- [ ] CONTRIBUTING 添加讨论指引

---
