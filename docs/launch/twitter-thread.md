### Tweet 1: 问题

```
问题很简单：

AI agents 很强大，但每次任务都是从零开始。

```

---

### Tweet 2: 解决方案

```
解决方案：强制能力沉淀

org-v2 的核心机制：
• 每个任务完成后必须评估"能否复用"
• 可复用的沉淀为 skill/data/knowledge
• 下次遇到类似问题，直接复用

简单粗暴，但有效。
```

### 预期问题

**Q: 为什么不用 X 框架？**
A: org-v2 专注能力沉淀，不是通用框架。可以与其他工具配合。
