# 社区能力示例

这里展示如何在实际项目中使用 org-v2 的能力管理系统。

## 📚 示例列表

1. **API 开发工作流** - RESTful API 标准开发流程
2. **数据库迁移工作流** - 生产数据库安全迁移
3. **代码审查检查清单** - PR 审查标准化流程

## 🎯 如何使用这些示例

1. **阅读示例**：了解完整工作流
2. **复制到项目**：将能力复制到你的 `capabilities/`
3. **按需调整**：根据团队实际情况修改
4. **持续改进**：使用后根据反馈优化

## 💡 贡献你的能力

欢迎分享你的能力！

1. Fork org-v2 仓库
2. 在 `docs/examples/community-capabilities/` 添加你的示例
3. 提交 Pull Request
4. 通过审查后合并
