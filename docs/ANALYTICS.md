# 分析与隐私

org-v2 收集简单的使用统计以改进产品。

---

## 📊 我们收集什么

### GitHub 内置指标（公开）
- Stars, Forks, Watchers
- Issues, PRs 数量
- Traffic（访问量）
- Clones（克隆数）
- Popular content（热门内容）

### 我们不收集
- ❌ 个人身份信息
- ❌ 代码内容
- ❌ 环境细节
- ❌ 使用行为追踪

---

## 🎯 数据用途

**仅用于**：
1. 了解项目使用情况
2. 识别流行功能
3. 发现改进方向
4. 分享里程碑成就

**不用于**：
- 个人识别
- 商业推广
- 第三方分享
- 任何侵犯隐私的行为

---

## 🔒 隐私保护

### 原则
- **透明**：明确告知收集内容
- **最小化**：只收集必要数据
- **匿名**：不关联个人身份
- **可选**：用户可选择退出

### GitHub Traffic
- GitHub 自动收集
- 14 天数据保留
- 聚合统计，无个人信息

### 本地统计
- 仅在 org-v2 repo 运行
- 不向外传输
- 保存在本地文件

---

## 🚫 如何退出

### 完全隐私使用
1. Fork 项目到私有仓库
2. 不 Star、不 Fork 公开 repo
3. 本地使用，不提交 Issues/PRs

### 部分退出
- 不参与 Discussions
- 不提交 Issues
- 只读文档
