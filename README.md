# AI-Native Org v2

一人公司 + AI agents 的极简运营系统。

**核心理念**：先跑起来，再沉淀能力。

## 🚀 30 秒上手

**今天做什么？** → 看 [NOW.md](NOW.md)

**按角色**：
- 👤 Owner → [control/owner.md](control/owner.md)
- 🤖 Agent → [ops/agent-protocol.md](ops/agent-protocol.md)
- 🔍 查能力 → [capabilities/](capabilities/)

## 📊 运行状态

- 活跃任务：0
- 能力资产：0
- 上周完成：0

自动更新：每周一 09:00

## 🎯 设计原则

1. **默认可见**：所有工作留 GitHub 痕迹
2. **能力优先**：每个任务问"能否复用"
3. **度量驱动**：无法度量 = 无法改进
4. **极简结构**：3 层足够（控制/运营/能力）

## 📁 结构

```
org-v2/
├── NOW.md              # 单一入口：当前工作台
├── control/            # Owner 控制层
├── ops/                # 日常运营
├── capabilities/       # 能力资产（skill/data/knowledge）
└── products/           # 产品索引（代码在独立仓库）
```

详细文档在各目录 README
